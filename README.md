# Portafolio

# Descripción

Este proyecto es una página web que comunmente se le llama portafolio que demuestra mi perfil profesional como desarrollador en programación, en este caso como desarrollador web; adjunto mi resumen, documentos, herramientas que domino y proyectos.

# Funcionalidad

La página se diseño para que fuera responsiva y adaptativa para diferentes resoluciones. El header es mi resumen con mi información que funciona por medio de flexbox para que pueda responder a la resolución de la pantalla y así pasar de fila a columna para mejor visualización con las resoluciones más pequeñas. El cuerpo o main muestra las herramientas que uso y los proyectos que realice como primer bloque, de la misma forma en que funciona el header se hizo uso de la flexbox para adaptar los elementos a razón de la variación de las pantallas y que la disposición sea comoda. El footer se muestra mi información para contactarme a traves de distintas redes y siendo más sencilla la disposición de los elementos usando un grid en los iconos de las redes.
